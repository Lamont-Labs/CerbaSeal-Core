CerbaSeal — System Brief (v0.1.0)

A one-page summary for reviewers, pilot decision-makers, and handoff. Print-friendly.

1One-line

CerbaSeal is a deterministic execution enforcement spine for AI-assisted workflows.

2What it does

  • Blocks unauthorized execution attempts
  • Enforces approval and control requirements
  • Produces verifiable evidence for every decision

3Core scenarios

  • AI attempts action → REJECT (AI_CANNOT_AUTHORIZE)
  • Missing approval → HOLD (REQUIRED_APPROVAL_MISSING)
  • Approved action → ALLOW (DECISION_ALLOWED)

4What it is NOT

  • Not a full governance platform
  • Not production deployed
  • Not independently security-reviewed
  • Not handling real client data

5Current status

Review-ready core. Not yet client deployed.
323 tests passing. Deterministic behavior verified.
